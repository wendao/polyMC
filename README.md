
** PolyMC ** a simple Monte Carlo code for multiple models

# Compiling and Installing
==============================
## Compile debug mode
    scons

## Compile release mode
    scons mode=release

## Clean debug mode
    scons -c

## Clean release mode
    scons -c mode=release
